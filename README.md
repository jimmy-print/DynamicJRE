# DynamicJRE
A command line podcast manager for The Joe Rogan Experience.

### Install
    git clone https://github.com/jimmy-print/DynamicJRE.git
    cd DynamicJRE
    pip install .

### Usage examples
    jrep --help
    jrep 1255
    jrep latest
I am not affiliated with JRE.
